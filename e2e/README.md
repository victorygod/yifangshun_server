# E2E 前端测试

使用 Playwright 进行端到端测试。

## 测试命令

```bash
# 运行所有测试
npm test

# 带 UI 运行（可视化调试）
npm run test:ui

# 有头模式（看到浏览器）
npm run test:headed

# 调试模式
npm run test:debug
```

## 测试文件

| 文件 | 说明 |
|------|------|
| login.spec.js | 登录页面测试 |
| admin.spec.js | 管理后台测试 |

## 测试覆盖

### 登录测试
- 页面正常加载
- 空手机号登录错误提示
- 无效手机号登录错误提示
- 默认超级管理员登录成功
- 登录成功后跳转

### 管理后台测试
- 首页正常加载
- 侧边栏菜单项存在
- 各模块切换（处方、库存、预约、用户）
- 表格搜索功能
- 新增/导出按钮

## 注意事项

1. 测试会自动启动服务器（如果未运行）
2. 测试使用 `home_super_admin` 账号登录
3. 测试数据使用本地 JSON 数据库，不影响生产环境
